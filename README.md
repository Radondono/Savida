# Savida

An open-source HTML-based interactive story engine maker.

## Features

- **Visual Node Editor** — Create branching stories with a drag-and-drop node graph
- **Character System** — Define characters with custom clothing, states, and body parts
- **Class Presets** — Save and reuse character templates
- **Image Support** — Upload base images and clothing layers for each character
- **Strip/Wear Mechanics** — Control clothing visibility per choice
- **Live Preview** — Test your story in a new tab with full image support
- **Export** — Export as standalone playable HTML or story.js data file
- **File System Saving** — Save directly to your computer with Ctrl+S
- **Drag & Drop** — Import .story.json files by dropping them on the window

## Getting Started

1. Open `index.html` in Chrome, Edge, or Opera
2. Click **📁 File → Set Save Folder** to choose where projects are stored
3. Create or import a project
4. Add scenes and connect them with the node editor
5. Press **Ctrl+S** to save, **Ctrl+P** to preview
